## **PRINT() FUNCTION**

```bash
#print a string
print("I am learning python")
```
`I am learning python`

```bash
#print any datatype
print(4+5)
```
`9`

```bash
#print multiple objects in one line
print("I love coading", 1+2, True)
```
`I love coading" 3 True`

```bash
#print multiple lines
print("I am a data scientist")
print(5+6)
print(False)
```
`I am a data scientist`
`11`
`False`
